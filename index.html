Post Bidding Item

Item to be bid.

This feature will proceed after the app has determined your item that it can sell you can now post the item for bidding and input your starting price. You can post up to 3 images of the item you are selling capturing it in different angles so the Junk Shop owners can examine the item that they can determine how much they will bid for the item.

Locator

Google Maps

This feature helps the Junk Shop to search for the location of a certain Household where the Junk Shop has to pick-up the items and buy it from them. This functionality will be of great use for the Junk Shop owners in order to locate certain Households in which they won the bidding. This will also help locate the nearest Household based on the location of the Junk Shop.

Live chat when you needed

Chat while you bid.

This feature helps the Junk Shop owners and househoulds to communicate while bidding is in the process if the Junk Shop owners ask for more details about the said item or more pictures the Households can upload another picture in the live chat area so that can satisfy the bidders.

Realtime Messaging service

Direct Messages

This feature will proceed after there is a winning Junk Shop bidder. The winning the Junk Shop owner can now directly message the household user for further more instructions in how/when the item will  be pick-up and pay by the Junk Shop owner.

Notification

Notifies the Household users and the winning Junk Shop owner.

These feature will procceed after there is a winning Junk Shop bidder. It will notify both the Junk Shop Owner and the Household users that there is a winning bidder and also notifies the Junk Shop owner that he has won the bidding proccess.
